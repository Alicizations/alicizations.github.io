在 CentOS 下并没有 sites-enabled/ 这个目录, 于是要先编辑 /etc/nginx/nginx.conf, 在 http block 中添加 include /etc/nginx/sites-enabled/*; 如图:

然后自己新建一个 sites-enabled/, 重启 nginx 服务(sudo systemctl restart nginx.service)就可以了
