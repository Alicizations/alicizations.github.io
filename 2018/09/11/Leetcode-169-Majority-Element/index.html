169.Majority Element

题目理解:

找出数组中出现次数大于 n/2 的数字

思路:

整体基于分治法的思想, 将数组分为三部分.

  • 1.随机选择一个数 v
  • 2.将数组分为 <v, =v, >v 三部分
  • 3.若 sizeof(=v) > n/2, 既满足要求, 则找到; 否则, 从剩下两个小数组中选取较大的一个重复步骤1 2 3

小结:

本周算法课刚好在讲分治法, 而刚好碰到类似的题, 便写了下来.
时间复杂度大概为 O(n)
